Overview

Cashfree is a popular payment gateway provider that offers a seamless and secure payment experience for Odoo customers. It allows customers to accept payments through various modes like credit/debit cards, UPI, net banking, and wallets. Integrating Cashfree with Odoo streamlines the checkout process, reduces transaction fees, and offers advanced features like payment reconciliation and fraud prevention, ultimately enhancing the customer experience and improving operational efficiency.

Features

This module provides you the facility to make payment through "Cashfree Payment".

Configuration

  • After installation of this module you will see the Cashfree in payment provider list.
  • Invoicing -> Configuration -> Payments -> Payment Providers.
  • Configure Cashfree for test environment.
  • Configure Cashfree for production environment.
  • Once you click on pay now button in odoo it will redirect to cashfree pay website. you can pay using any of this payment method with use of cashfree pay.
